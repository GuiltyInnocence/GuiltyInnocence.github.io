TRUE TEARS 短评

2018-11-16

-181113

本来说先看物语系列的结果还是拗不过TARI TARI里这一波安利来看真实之泪了。。。

就因为tari tari里合唱的那首是这部的OP而已。。

本作放映于2008年，于是你可以找到很多鲜明的时代印记:第一集就有女主的蓝白胖次，然后这个画风也是很有年代感(原来当年的啪社也是这样的画风emmm)。

然后op演唱enfonius，ed畑妈作词总有种。。

追着冈妈早期作品来的，顺便说一句还是啪社这套人马(冈妈，东地，关口)。声优倒是坂井和奏高恒彩阳

顺便吐槽一句啪社有很多万恶之源，比如花开里的nako是nagiasu前身，TARI TARI里的sawa是赛马娘前身(大误)

-181114

第三集。

就算我是看惯了冈妈世界观的人，也不能接受这么突然的剧情进展啊前情解释太少了啊

第三集开始冈妈式贵圈真乱发展。主要看不懂的就是男主和hiromi之间的关系究竟是什么样，根本把握不住冈妈风格嘛

181115

第五集。

感觉剧情越来越扑朔迷离了。。
不过基本线索还是能抓住的。

本集中间出现了视角转换的部分(为什么会想到漫无止境的八月)。随即突然就来了震撼的剧情:男主和汤浅家的合影。这时候男主母亲为什么这么针对hiromi基本真相大白了，但两家的关系纠葛恩怨情仇才刚刚开始(怎么感觉跟国产家族争斗剧一副德行。。怪不得五年以后冈妈能写出nagiasu这种狗血剧情)。

另外说回男主和noe之间感觉还是蛮正经的交往emmm

这部确实适合和TARI TARI放在一起谈论，从声优到op neta到风格(虽然啪社风格都很像就是了)，而且回头看看啪社的人物画风在这时候基本有雏形了(感觉是花开伊吕波正式成型)，特别是看配角人物画风特别明显。

本作的背景好像并没有东地参与，于是出现了很有意思的一种画风——类似版画或油画的作画风格(看背景部分比较明显，画面四角有时还有模糊一样的效果)。啪社画风的过渡阶段

这里开始男主看出问题来了。

之前的分析基本忽略了爱子，但这集主要矛盾和次要矛盾已经发生变化(深受政治毒害的高二狗在此)

完了这里出来了是真的不对头了
